## 🚀 **Deployment on Render**

### **Prerequisites**
- GitHub repository: `https://github.com/RCAS-InnovationandStartupCell/RCAS-Innovation-Cell-Magazine-2026`
- Render account (free tier available)

### **Deployment Steps**

1. **Push to GitHub**
   ```bash
   git add .
   git commit -m "Deploy RCAS Innovation Cell Magazine 2026"
   git push origin main
   ```

2. **Setup on Render**
   - Go to [render.com](https://render.com)
   - Click "New +" → "Web Service"
   - Connect your GitHub repository
   - Select the repository
   - Render will automatically detect the `render.yaml` configuration

3. **Configuration**
   - **Service Name**: `rcas-magazine-2026`
   - **Environment**: `Static`
   - **Build Command**: `echo 'No build needed for static site'`
   - **Publish Directory**: `.`
   - **Branch**: `main`

4. **Deploy**
   - Click "Create Web Service"
   - Render will automatically deploy your static site
   - Your site will be available at: `https://rcas-magazine-2026.onrender.com`

### **Automatic Deployments**
- **GitHub Integration**: Every push to main branch triggers automatic redeployment
- **Zero Downtime**: Render handles deployment with zero downtime
- **SSL Certificate**: Automatically provided
- **CDN**: Global CDN for fast loading
